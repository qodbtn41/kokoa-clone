State

  • :active - 클릭된 상태
  • :hover - 마우스가 올라가있는 상태
  • :focus - input에 커서가 가있는 상태.
  • :visited - a 태그에만 적용
  • :focus-within - 자식이 포커스 된 경우
네이버

부모의 상태에 따른 자식의 스타일 변경

"부모:hover 자식" - 부모가 hover된 자식을 선택하는 Selector

자식의 상태에 따른 부모의 스타일 변경

focus-within을 사용한다.

부모와 자식의 상태에 따라서 스타일 변경

"부모:hover input:focus" - 부모가 hover된 자식 input 중 focus된 경우
